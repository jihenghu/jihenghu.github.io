NCL 绘制一维数据的contour图

参考来源，【官方文档】

If you have X, Y, Z data represented by one-dimensional (1D) arrays of the same length, then NCL will contour this data by first generating a triangular mesh of the data under the hood, and then contouring the triangular mesh. You can also choose to regrid or interpolate your data to a 2D grid before plotting.
Here are the options for contouring 1D data:
Set the sfXArray and sfYArray resources to the X and Y arrays respectively, and pass the 1D Z data to the appropriate gsn_csm contouring routine.
If your X/Y arrays are lon/lat arrays, then in NCL V6.4.0 and later you can attach the special "lat1d" / "lon1d" attributes to your data variable to be plotted. This effectively causes the sfXArray and sfYArray resources to be set for you under the hood.
See the "Plotting data on map" examples page for more information.
If you have arrays representing polygons that surround each data point, then you can additionally set sfXCellBounds and sfYCellBounds to these arrays for a (potentially) better plot.
You can first regrid or interpolate the data to a rectilinear, or curvilinear grid using ESMF regridding, or functions like like cssgrid, natgrid and triple2grid.
See the "Random to Grid" examples page.
In all cases, the quality of the resulting plot will be a function of the distribution of the X and Y arrays, the number of sampling points, and the 'shape' of the data be contoured.

实际上一维变量是可以使用contour类函数绘制等值线填充图的，只需要设置sfYArray 和sfXArray 属性来指定对应的经纬度，但是在凹多边形区域会出现拉丝现象"streaking effect"
为了消除这个现象，必须指定每点的多边形边界属性sfXCellBounds、sfYCellBounds

示例：

begin

mlsefile="../data/mlse.cn.tbv12.myd06.era20c.res1.0_0001_improved_core.v2020.20080426.txt"
nrow=numAsciiRow(mlsefile)
ncol=numAsciiCol(mlsefile)
fio=asciiread(mlsefile,(/nrow,ncol/),"float")
lon=fio(:,1)
lat=fio(:,2)
stime=fio(:,3)
land=fio(:,7)
cwp=fio(:,32:34)
cwp@units="g.m~S~-2~N~"
cfr=fio(:,44:46)
cfr@units="%"
cwp=where(cwp.eq.0,cwp@_FillValue,cwp)
cfr=where(cfr.eq.0,cfr@_FillValue,cfr)

edgx=new((/nrow,3/),"float")
edgy=new((/nrow,3/),"float")
dx=0.075
edgx(:,0)=lon
edgx(:,1)=lon-dx*1.732
edgx(:,2)=lon+dx*1.732
edgy(:,0)=lat+dx*2
edgy(:,1)=lat-dx
edgy(:,2)=lat+dx

wks=gsn_open_wks("png","plot.cfr.cwp")
gsn_define_colormap(wks,"NCV_bright")
res=True
res@gsnFrame=False
res@gsnDraw=False
res@cnFillOn=True
res@cnLinesOn = False
res@cnLineLabelsOn = False
res@cnInfoLabelOn = False
res@cnFillMode = "CellFill"
res@lbLabelBarOn   =True;

res@gsnAddCyclic = False 
res@mpMinLatF = 10
res@mpMaxLatF = 55
res@mpMinLonF = 70	
res@mpMaxLonF = 145

res@mpFillOn = True
res@cnFillDrawOrder      = "PostDraw"
res@mpLandFillColor      = "grey"
res@mpOceanFillColor     = "lightblue"  
res@mpOutlineOn = True
; res@mpOutlineBoundarySets = "National" 
res@mpOutlineSpecifiers = "China"  	 
res@mpGeophysicalLineThicknessF = 1.0     
res@mpNationalLineThicknessF = 1
res@mpDataBaseVersion = "MediumRes"             
res@mpDataSetName = "Earth..4"      
; res@mpAreaMaskingOn = True
; res@mpMaskAreaSpecifiers = (/"China:states","Taiwan"/)   

;;--- tickmark	 
res@tmXBMode = "Explicit"
res@tmYLMode = "Explicit"
res@tmXBLabels = (/"80"+"~S~o~N~E","","100"+"~S~o~N~E","","120"+"~S~o~N~E","","140"+"~S~o~N~E"/)
res@tmXBValues = (/80,90,100,110,120,130,140/)                                                   
res@tmYLLabels = (/10,20,30,40,50/)+"~S~o~N~N"
res@tmYLValues = (/10,20,30,40,50/)
c = 0.013                                                        
res@tmYLMajorLengthF = 0.008
res@tmXBMajorLengthF = 0.008
res@tmXBLabelFontHeightF = 0.018                                              
res@tmYLLabelFontHeightF = 0.018 
res@sfXArray  = lon
res@sfYArray = lat
res@sfXCellBounds =edgx
res@sfYCellBounds =edgy
res@cnLevelSelectionMode  = "ExplicitLevels"   
res@cnLevels =  (/1,5,10,20,30,50,80,100,200,500,1000/) 
res@cnMissingValFillColor="white"

res@lbLabelFontHeightF = 0.022
res@lbBoxEndCapStyle = "TriangleBothEnds"
res@pmLabelBarWidthF = 0.7
res@gsnLeftStringFontHeightF =0.03	  
res@gsnRightStringFontHeightF =0.03	  
plot=new(6,graphic)
title="("+(/"a","b","c","d","e","f"/)+")"
phase=(/"liquid","ice","mixed"/)
do i=0,2
	res@gsnLeftString=title(i)+" CWP ("+phase(i)+" phase)"
	plot(i) = gsn_csm_contour_map(wks,cwp(:,i), res) 
end do
res@cnLevels := (/10,20,30,40,50,60,70,80,90/) 
do i=3,5
	res@gsnLeftString=title(i)+" Cloud Fraction ("+phase(i-3)+" phase)"
	plot(i) = gsn_csm_contour_map(wks,cfr(:,i-3), res) 
end do
resP=True
resP@gsnPanelMainString= "MODIS Clouds on 26~S~th~N~ Apr,2008"  
resP@gsnPanelMainFontHeightF= 0.015 
resP@gsnPanelMainFontThicknessF= 0.01 
gsn_panel(wks,plot,(/2,3/),resP)
end
