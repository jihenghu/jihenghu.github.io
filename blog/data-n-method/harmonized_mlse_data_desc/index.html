Data Description for Daily Records of Harmonized MLSEs

Created2025-12-28|Updated2025-12-29|Data N' Method
|Word count:529|Reading time:2min|Post View:

参数说明

iscan,ipixel：该观测在当前轨道（原始L1级产品）中的行列号

nscan,npixel: 所在原始网格的行数和列数
基本上你可以用这四个量和lon, lat 还原出原始的二维网格

lon, lat, stime, lst, t2m, snowc, smc, lza, seafrac
经纬度，UTC时间（十进制），地表温度，2m温度，积雪百分比，土壤湿度，静止卫星天顶角，海洋面积百分比

cfr(3), clear(3), missing(3)
三个分辨率等级像斑下的云分数，晴空分数，缺测分数

cldflag： 所使用的静止卫星标号

emiss,tb : 原始emissivity, 亮温

amsrx(7) ：xcalibrated的AMSR2 emissivity: 10v, 10h, 19v, 19h, 23v, 36v, 36h

mwrix(9) : xcalibrated的MWRI emissivity: 10v, 10h, 19v, 19h, 23v, 36v, 36h, 89v, 89h
